# Ícones PWA

Este diretório deve conter os seguintes arquivos PNG:

- `icon-192x192.png` - Ícone 192x192 pixels
- `icon-512x512.png` - Ícone 512x512 pixels

Você pode:
1. Usar um gerador online de PWA icons
2. Converter o `icon.svg` fornecido para PNG nos tamanhos necessários
3. Criar seus próprios ícones personalizados

Para desenvolvimento, o app funcionará sem os ícones, mas para produção é recomendado adicionar ícones reais.
